# 从微信跳一跳学习逆向开发 iOS

[TOC]

### 一、libimobiledevice与 adb

libimobiledevice：逆向出 iOS 与 Mac Windows 接口的通讯协议

adb：安卓下调试工具



### 二、安装libimobiledevice

```
$ brew update
$ brew install libimobiledevice
#libimobiledevice中并不包含ipa的安装命令，所以还需要安装
$ brew install ideviceinstaller
```
